Listas de terminos

HTML
HTML (Lenguaje de Marcas de Hipertexto,
del inglés HyperText Markup Language) es el componente más básico de la Web.
Define el significado y la estructura del contenido web.
CSS
Hojas de Estilo en Cascada (del inglés Cascading Style Sheets)
o CSS es el lenguaje de estilos utilizado para describir la presentación
de documentos HTML o XML (en-US) (incluyendo varios languages basados en XML como SVG, MathML o XHTML).
CSS describe como debe ser renderizado el elemento estructurado en la pantalla, en papel, en el habla o en otros medios.
